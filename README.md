# Bootstrap - Shalom Harlow

Este proyecto es una página web sencilla dedicada a la supermodelo Shalom Harlow, utilizando [Bootstrap](https://getbootstrap.com/) para el diseño responsivo y estilos personalizados.

## Estructura del proyecto

- `index.html`: Página principal con tarjetas informativas sobre Shalom Harlow.
- `estilos/estilo.css`: Estilos personalizados para mejorar la apariencia visual.
- `package.json`: Configuración del proyecto y dependencias (Bootstrap).
- `.gitignore`: Ignora la carpeta `node_modules`.

## Instalación

1. Clona este repositorio.
2. Instala las dependencias ejecutando:

   ```sh
   npm install
   ```

3. Abre `index.html` en tu navegador.

## Tecnologías utilizadas

- [Bootstrap 5](https://getbootstrap.com/)
- HTML5
- CSS3
